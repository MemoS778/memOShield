Temel Yetenekler

IDS — Saldiri Tespit Sistemi

Ag trafigini gercek zamanli analiz ederek DoS/DDoS, port taramasi ve brute-force saldirilari otomatik olarak tespit eder ve mudahale eder.

  • Kayan pencere algoritmasi ile davranis analizi
  • Esik tabanli otomatik ban mekanizmasi
  • Whitelist destegi ile guvenilir IP korunmasi
  • 50ms altinda tepki suresi

Honeypot — Aldatma Mekanizmasi

Sahte FTP, Telnet ve MySQL servisleri olusturarak saldirganlari tuzaga dusurur; gercek altyapiyi korur ve istihbarat toplar.

  • 3 farkli sahte servis (port 2121, 2323, 3307)
  • Baglanan IP otomatik olarak engellenir
  • GeoIP ile saldirgan profilleme
  • Tum etkilesimler veritabanina kaydedilir

GeoIP — Cografi Konum Analizi

Tehdit kaynaklarini harita uzerinde gercek zamanli gorsellestirerek ulke, ISP ve hostname bilgilerini belirler.

  • Canli interaktif dunya haritasi (Leaflet.js)
  • Ulke ve bolge bazli istatistikler
  • ISP ve organizasyon tespiti
  • Konum bazli alarm ve raporlama

PCAP — Adli Bilisim Kaydi

Supheli paketler otomatik olarak PCAP formatinda kaydedilir; Wireshark ile derinlemesine analiz yapilabilir.

  • Otomatik paket kaydi baslatma
  • Wireshark uyumlu cikti formati
  • Forensik analiz icin arsivleme
  • Olay kaniti olarak kullanilabilir

SSE — Gercek Zamanli Akis

Server-Sent Events protokolu ile dashboard'a anlik olay akisi saglanir; sayfa yenilemeden canli izleme yapilabilir.

  • Milisaniye seviyesinde olay iletimi
  • Dusuk bant genisligi tuketimi
  • Otomatik yeniden baglanti
  • Coklu istemci destegi

Dinamik Firewall

Linux sistemlerde iptables/nftables kurallari otomatik olarak guncellenir; tehdit algilanan IP adresleri aninda engellenir.

  • iptables ve nftables backend destegi
  • Otomatik kural ekleme ve kaldirma
  • Veritabani tabanli kural kaliciligi
  • Windows simulasyon modu

Sistem Mimarisi

Go Core Engine (Paket Yakalama)

gopacket/pcap kutuphanesi ile ham paketleri yakalayarak kaynak IP, hedef port ve protokol bilgilerini cikarir. Kayan pencere algoritmasi ile her IP'nin davranisini izler.

Go SSE Broker (Olay Dagitimi)

Core Engine'den gelen olaylari alir, Bearer token ile dogrulama yapar ve SSE protokolu uzerinden tum bagli dashboard'lara aninda yayinlar.

Flask Dashboard (Arayuz)

Chart.js grafikleri, Leaflet haritasi ve canli olay tablosu ile tum tehditleri gorsel olarak sunar. Yonetici paneli uzerinden IP banlama ve sorgulama islemleri yapilabilir.

IDS Modulu (Tespit)

Her IP icin zaman damgali kuyruk tutar; esik degeri asildiginda firewall kurali ekler, veritabanina yazar ve bildirim tetikler.

Bildirim Sistemi

Kritik olaylar tespit edildiginde Telegram Bot API ve Slack Webhook uzerinden otomatik alert gonderimi yapar.

Veritabani ve Raporlama

Tum olaylar SQLite veritabaninda kalici olarak saklanir. PDF rapor uretimi ve PCAP arsivleme ile forensik analiz desteklenir.

Teknik Detaylar

Programlama Dilleri

  • Backend: Python 3.10+ (Flask framework)
  • Paket Motoru: Go (gopacket/pcap)
  • SSE Broker: Go (net/http)
  • Frontend: HTML5, CSS3, Vanilla JavaScript

Performans Degerleri

  • Tepki Suresi: <50ms (algilama → engelleme)
  • DoS Esigi: 200+ paket/saniye
  • Port Tarama Esigi: 20+ port / 10 saniye
  • Honeypot Portlari: FTP(2121), Telnet(2323), MySQL(3307)

Veri Saklama

  • Veritabani: SQLite (yerel dosya tabanli)
  • PCAP Kayit: Otomatik dosya arsivleme
  • Olay Gecmisi: Son 200 olay dashboard'da gorunur
  • Kural Kaliciligi: Veritabani tabanli, restart sonrasi korunur

Entegrasyon ve API

  • REST API: /api/events, /api/ban, /api/bans
  • Bildirim: Telegram Bot API, Slack Webhook
  • GeoIP: ipapi.co harici servisi
  • Gorsellestirme: Chart.js, Leaflet.js

Guvenlik Mekanizmalari

  • Kimlik Dogrulama: Session tabanli admin girisi
  • Hizmetler Arasi: Bearer token dogrulama
  • TLS Destegi: Sertifika ile sifreleme (opsiyonel)
  • Whitelist: Guvenilir IP'ler ban'dan muaf

Dagitim Secenekleri

  • Yerel: Python venv + dogrudan calistirma
  • Docker: Dockerfile ile konteyner
  • Sunucu: systemd + gunicorn (Ubuntu)
  • Windows: Simulasyon modu (iptables haric)

Saldiri Senaryosu

1. Kesif

Saldirgan, hedef sunucudaki acik portlari taramaya baslar.

2. Tuzak

Honeypot portlarina (sahte FTP/MySQL) baglanir, tuzaga duser.

3. Istihbarat

Go Core IP'yi yakalar; GeoIP ile ulke, ISP ve hostname belirlenir.

4. Engelleme

Sistem 50ms icinde iptables kuralini ekler, IP engellenir.

5. Kayit

Olay veritabanina yazilir, dashboard'da alarm belirir, bildirim gonderilir.
